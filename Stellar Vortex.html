Driving function 
E_perm ← E_perm + 0.35       (per click, never decays)
E_tran ← E_tran + 0.7        (transient, γ = 1.5)
E      =  tanh(E_perm · 0.22) · 4   +  E_tran
ω(r)   = (0.55 + 0.30·E) + 1.0/(r + 0.2)
θ′     = θ + ω(r) · t  +  0.4·E
ρ(p)   = fbm( R(θ′) · p · 3 )
bg     = lerp(∈0, ∈grey, y) + 0.022·fbm(·)   (monochrome)
rim    = vec3(0.055 + 0.06·fbm(p·0.42 + t))   (mesh, was indigo)
Energy is split into two accumulators: a permanent baseline that never decays (each click ratchets it up), and a fast transient for the per-tap kick. The permanent term is tanh-squashed so the visual saturates gracefully past ~30 clicks rather than blowing out. Effectively: clicks can only ever make the vortex bigger. Cursor is a directional light: p̂ · m̂ brightens the hemisphere facing the pointer. Background is a near-black luminance gradient with a slow large-scale fBM shimmer so empty frames still breathe.
